# Assets Folder

This folder contains static assets for the portfolio website.

## Structure

```
assets/
├── images/          # Profile images, project screenshots
└── resume.pdf       # Your resume/CV (add your file here)
```

## Instructions

1. **Profile Image**: Add your professional photo as `profile.jpg` or `profile.png` in the `images/` folder
2. **Project Images**: Add project screenshots/thumbnails in the `images/` folder
3. **Resume**: Add your resume as `resume.pdf` in this folder (referenced in the "Download CV" button)

## Image Guidelines

- **Profile Photo**: 
  - Square aspect ratio (1:1)
  - Minimum 500x500px
  - Professional quality
  
- **Project Images**:
  - 16:9 or 4:3 aspect ratio
  - Minimum 800px width
  - Compressed for web (< 500KB each)

## File Naming Convention

Use lowercase with hyphens:
- ✅ `my-project-name.jpg`
- ❌ `My Project Name.jpg`
